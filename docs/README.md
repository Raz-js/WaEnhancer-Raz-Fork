# App Template

This repository is a neutral Android app template scaffold. It provides a starting structure with example resources, a sample activity, and Gradle configuration to kick off a new Android project.

## Purpose

Make it easy to start a new app by providing a minimal, configurable base: change the app name, add/remove dependencies, create flavors, and customize resources.

## How to use this template

1. Update `settings.gradle.kts` `rootProject.name` with your project name.
2. Update `app/src/main/res/values/strings.xml` `app_name` to set the display name.
3. Edit `app/build.gradle.kts` to add the libraries you need.
4. Remove any example code or assets you don't want and start developing.
